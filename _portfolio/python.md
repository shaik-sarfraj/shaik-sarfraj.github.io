## Overview

A Python-based urban data analysis project developed as part of the Programmable Cities course at UNSW Sydney, focusing on applying computational methods to analyse and visualise spatial patterns.

The project demonstrates the use of Python programming for transforming raw urban datasets into meaningful analytical outputs through data processing, spatial interpretation, and visual communication techniques.

By combining programming workflows with urban analytics concepts, the project explores how computational approaches can support the understanding of complex urban phenomena and improve evidence-based analysis.

<img src="/assets/images/projects/python/heatmap-1.png" alt="Python Heat Map Analysis">

## Methodology

The workflow involved:

- Importing, cleaning, and structuring urban datasets for analysis.
- Applying Python-based data processing techniques to prepare datasets for visual exploration.
- Conducting analytical operations to identify spatial and temporal patterns.
- Developing heat map visualisations to communicate variations and relationships within urban datasets.
- Interpreting analytical outputs to understand underlying urban trends.

The project highlights the role of Python as a tool for reproducible urban analytics workflows, enabling efficient processing and communication of complex datasets.

# Urban Data Analysis — Heat Map Visualisation

## Overview

Developed as part of the Programmable Cities course at UNSW Sydney, this project applies Python programming techniques to analyse and visualise urban datasets.

The project focuses on transforming raw data into interpretable analytical outputs through data processing, spatial exploration, and visual communication.

By integrating computational methods with urban analytics concepts, the workflow demonstrates how programming can support the investigation of complex urban patterns.

<img src="/assets/images/projects/python/heatmap-1.png" alt="Python Heat Map Analysis">


## Methodology

The workflow involved:

- Importing, cleaning, and structuring urban datasets for analysis.
- Applying Python-based data processing techniques to prepare datasets.
- Conducting analytical operations to identify spatial and temporal patterns.
- Developing heat map visualisations to communicate analytical findings.
- Interpreting outputs to understand relationships within urban datasets.

# Environmental Data Analysis — Urban Wind Speed Modelling

## Overview

This project investigates urban environmental conditions through Python-based data analysis and visualisation workflows.

The study examines variations in wind speed around urban configurations using high-resolution numerical simulation datasets and demonstrates how computational methods can support environmental performance assessment.

The workflow highlights the application of programming techniques for processing large datasets, identifying patterns, and communicating environmental relationships.

<img src="/assets/images/projects/python/windspeed-main.png" class="project-main-image" alt="Python Wind Speed Analysis">

## Methodology

The workflow involved:

- Preparing and processing environmental datasets for analysis.
- Applying Python-based data manipulation techniques.
- Conducting statistical exploration of wind speed variations.
- Developing graphical outputs to communicate environmental trends.
- Interpreting relationships between urban form and environmental performance.

<img src="/assets/images/projects/python/windspeed-1.png" alt="Python Wind Speed Analysis">

## Key Outputs

The project generated:

- Visual representations of wind speed variations.
